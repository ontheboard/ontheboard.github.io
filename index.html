Chessix

Chessix is aLos Alamos style Minichess variant, played on a six by six square board, programmed in Lua for the TIC-80 environment.

Rules for this minichess variant are the same as for normal chess, except that there is no castling, no double pawn move, and no taking en-passant. Chessix deviates from the classic Los Alamos variant in that it allows you to promote to bishops - as well as queens, rooks, and knights.

Limitations of Chessix

Chessix is running in an emulated environment, using Lua code, on top of a browser that uses JavaScript, and as such it will be slower than many chess programs.

Chessix does not use neural networks, and as such will not present a robust tactical challenge, nor have whatever advantage of speed comes from using neural networks.

TIC-80 does not make available commands for export of data, because of which Chessix does not allow saving or export of games.
